Game Description

This project was a point and click adventure game. The user would play as the character Little Red, who must travel through the woods to help her sick grandmother. She faces many survival challenges, including finding food, staying warm at night and fighting the big bad wolf.

First scene

Collaborative Piece

Little Red was a collaborative piece between myself and Donald Hoffer. Donald coded all of the JavaScript to make the piece functional, while I focused on designing the graphics and user interface, as well as developing the overall concept.

Graphic Elements

The imagery are examples of the various scenes the user travels through, as well characters and items one can interact with during the game. All of the artwork has a consistent style: simple with flat colors.

Characters and collectible items

User Testing and Problem Solving

The development of this game went through several revisions as we had many users provide their feedback after playing for a few rounds.

This feedback included: revising the game directions to explain the rules more clearly, adjusting the random number generation and the chances of success and adding additional interactive features to help keep the users engaged.

Overall, we feel the game was a success as many players enjoyed the challenge, and we were able to learn a great deal about user interaction.
